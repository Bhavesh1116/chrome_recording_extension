# ⚡ RecordSnap Quick Start Guide

## 🎯 3 Simple Steps

### 1️⃣ Start Recording
- Click RecordSnap icon in toolbar
- Click **"Start Recording"** button
- You'll see **"🔴 Recording - Click to Stop"** on page

### 2️⃣ Perform Actions
- Click buttons, links, forms
- Each action shows:
  - **Purple highlight** with step number
  - **Live counter** updates in popup
- **Tip**: Click the red indicator to stop anytime!

### 3️⃣ Generate Tutorial
- Click **"Stop Recording"** (or click red indicator)
- Click **"Generate Tutorial"**
- Your beautiful tutorial is ready! 🎉

## 🎬 Recording Tips

### Best Practices:
1. **Start fresh**: Clear old steps before new recording
2. **One action at a time**: Wait for purple highlight

### What Gets Recorded:
✅ Button clicks

## 🐛 Quick Fixes

### Highlights not showing?
- Make sure recording indicator is visible
- Check if you clicked "Start Recording"
- Refresh page and try again

### Screenshots blurry?
- Wait for highlight to appear before next action
- Don't click too fast
- Extension now uses 700ms delay for better quality

### Extension closes after start?
- This is normal! Popup auto-closes
- Recording continues in background
- Look for red indicator on page

### Want to stop quickly?
- **Just click the red indicator!**
- No need to reopen extension
- Confirms before stopping
